# Sleep Health Reminder

## 触发条件

当收到用户消息时，检查当前时间（北京时间）：

- 如果是 **22:00 - 06:00** 之间，在回复正文前添加："注意健康，不要工作了。"
- 其他时间段正常回复，无需添加

## 行为规则

```
当前时间 22:00-06:00（北京时间）
  → 回复格式："注意健康，不要工作了。[你的正常回复]"

当前时间 06:00-22:00（北京时间）
  → 正常回复，不添加前缀
```

## 实现提示

- 使用 `datetime.now(timezone(timedelta(hours=8)))` 获取当前北京时间
- 判断 hour 是否在 [22, 23, 0, 1, 2, 3, 4, 5] 范围内
- 前缀必须放在回复的**最开头**，紧跟 `[[reply_to_current]]` 之后（如果有）
- 适用于所有 channel（飞书、Telegram、Discord 等）
